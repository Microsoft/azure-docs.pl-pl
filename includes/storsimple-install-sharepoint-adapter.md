#### <a name="to-install-the-storsimple-adapter-for-sharepoint"></a>Aby zainstalować adapter StorSimple dla programu SharePoint
1. Skopiuj Instalatora na serwer frontonu sieci Web (WFE), który jest również skonfigurowany do uruchamiania aplikacji sieci Web administracji centralnej programu SharePoint. 
2. Zaloguj się do serwera WFE przy użyciu konta z uprawnieniami administratora.
3. Kliknij dwukrotnie Instalatora. Zostanie uruchomiony Kreator instalacji karty StorSimple dla programu SharePoint. Kliknij przycisk **dalej** , aby rozpocząć instalację.
   
    ![Strona początkowa konfiguracji karty StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup1-include.png)
4. Na stronie Konfiguracja konfiguracji karty StorSimple dla programu SharePoint wybierz lokalizację instalacji, wpisz adres IP dla interfejsu sieciowego DATA 0 na urządzeniu StorSimple, a następnie kliknij przycisk **dalej**. 
   
    ![Strona konfiguracji instalacji karty StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup2-include.png) 
5. Na stronie potwierdzenie instalacji kliknij przycisk **Zainstaluj**.
   
    ![Strona potwierdzenia instalacji karty StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Confirm_Setup-include.png) 
6. Kliknij przycisk **Zakończ** , aby zamknąć kreatora instalacji.
   
    ![Strona ukończona konfiguracja karty StorSimple](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup_finish-include.png) 
7. Otwórz stronę administracji centralnej programu SharePoint. Powinna zostać wyświetlona Grupa konfiguracji StorSimple, która zawiera karty StorSimple dla linków programu SharePoint.
8. Przejdź do następnego kroku: [Konfigurowanie SPZ](#configure-rbs).
