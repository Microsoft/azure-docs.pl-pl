## <a name="go-to-your-api-management-instance"></a>Przechodzenie do wystąpienia usługi API Management

1. W Azure Portal wyszukaj i wybierz pozycję **API Management usługi**.

    :::image type="content" source="media/api-management-navigate-to-instance/view-apim-1.png" alt-text="Wybierz API Management usługi":::

1. Na **stronie API Management usług** wybierz swoje API Management usługi.

    :::image type="content" source="media/api-management-navigate-to-instance/view-apim-2.png" alt-text="Wybierz swoje API Management wystąpienia":::
