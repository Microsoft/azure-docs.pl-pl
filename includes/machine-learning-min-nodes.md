> [!IMPORTANT]
> Aby uniknąć naliczania opłat, gdy nie są uruchomione żadne zadania, należy **ustawić minimalną wartość równą 0**. To ustawienie umożliwia Azure Machine Learning cofnięcia alokacji węzłów, gdy nie są one używane. Każda wartość większa niż 0 będzie utrzymywać tę liczbę uruchomionych węzłów, nawet jeśli nie są one używane.
