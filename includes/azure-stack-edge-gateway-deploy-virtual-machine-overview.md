Korzystając z interfejsów API, można tworzyć maszyny wirtualne i zarządzać nimi na urządzeniu usługi Azure Stack Edge. Te interfejsy API są standardowymi interfejsami API Azure Resource Manager, które są wywoływane przy użyciu lokalnego punktu końcowego Azure Stack Edge. Interfejsy API usługi Azure Resource Manager zapewniają spójną warstwę zarządzania, która w tym przypadku umożliwia tworzenie, aktualizowanie i usuwanie maszyn wirtualnych w lokalnej subskrypcji, która istnieje na urządzeniu. Można nawiązać połączenie z usługą Azure Resource Manager uruchomioną w usłudze Azure Stack Edge za pomocą poleceń cmdlet programu Azure PowerShell.
