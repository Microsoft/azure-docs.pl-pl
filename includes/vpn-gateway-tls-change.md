Od 1 lipca 2018 r. z usługi Azure VPN Gateway zostanie usunięta obsługa techniczna protokołów TLS 1.0 i 1.1. Usługa VPN Gateway będzie obsługiwać tylko protokół TLS 1.2. Wpływ dotyczy tylko połączeń punkt-lokacja; nie wpłynie to na połączenia lokacja-lokacja. Jeśli używasz protokołu TLS do połączeń sieci VPN typu punkt-lokacja na klientach z systemem Windows 10, nie musisz podejmować żadnych działań. Jeśli używasz protokołu TLS do połączeń punkt-lokacja na klientach z systemem Windows 7 i Windows 8, zapoznaj się z instrukcjami dotyczącymi [VPN Gateway często zadawanych pytań](../articles/vpn-gateway/vpn-gateway-vpn-faq.md#P2S) dotyczących aktualizacji.
