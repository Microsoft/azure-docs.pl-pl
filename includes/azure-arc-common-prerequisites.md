## <a name="prerequisites"></a>Wymagania wstępne

Przed rozpoczęciem wykonywania zadań w tym artykule należy wykonać następujące czynności:

- [Zainstalowane narzędzia klienta](../articles/azure-arc/data/install-client-tools.md)

   Narzędzia obejmują `azdata` , Azure Data Studio i interfejs wiersza polecenia platformy Azure.

   Aby uzyskać Azure Data Studio, zainstaluj rozszerzenia dla **interfejsu wiersza polecenia danych platformy Azure**, **usługi Azure Arc** i **PostgreSQL** (w zależności od usług danych).

- [Zainstalowano kontroler danych usługi Azure Arc](../articles/azure-arc/data/create-data-controller.md)
