# <a name="azure-security-baseline-for-customer-lockbox-for-microsoft-azure"></a>Podstawowa usługa Azure Security dla Skrytka klienta dla Microsoft Azure

Podstawowa usługa Azure Security dla Skrytka klienta dla Microsoft Azure zawiera zalecenia, które pomogą ulepszyć stan bezpieczeństwa wdrożenia.

Punkt odniesienia dla tej usługi jest rysowany w [wersji 1,0 usługi Azure Security test](../benchmarks/overview.md), która zawiera zalecenia dotyczące sposobu zabezpieczania rozwiązań w chmurze na platformie Azure z naszymi najlepszymi wskazówkami.

Aby uzyskać więcej informacji, zobacz [podstawy zabezpieczeń platformy Azure — omówienie](../benchmarks/security-baselines-overview.md).
