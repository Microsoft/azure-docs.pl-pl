# <a name="azure-storage-archive-backup-and-disaster-recovery-partners"></a>Archiwum, kopia zapasowa i partnerzy odzyskiwania po awarii usługi Azure Storage

W tym artykule przedstawiono partnerów firmy Microsoft zintegrowanych z usługą Azure Storage w celu archiwizacji, tworzenia kopii zapasowych oraz obciążeń związanych z ciągłością biznesową i odzyskiwaniem po awarii (BCDR). Te rozwiązania partnerskie wykorzystują korzyści ze skalowania i kosztów usługi Azure Storage. Możesz użyć rozwiązań, aby pomóc w rozwiązywaniu problemów z tworzeniem kopii zapasowych, utworzyć lokację odzyskiwania po awarii lub archiwizować nieużywaną zawartość do długoterminowego przechowywania. Wszystkie standardy zgodności spełniające wymagania usługi Azure Storage oraz funkcje platformy Azure, takie jak [niezmienne Zarządzanie magazynem](../../../blobs/storage-blob-immutable-storage.md) i [cyklem życia](../../../blobs/storage-lifecycle-management-concepts.md), umożliwiają łatwe zastępowanie kopii zapasowych opartych na taśmach i oferowanie oszczędnej witryny odzyskiwania na żądanie.
