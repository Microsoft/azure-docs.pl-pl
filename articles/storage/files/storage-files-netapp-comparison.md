# <a name="azure-files-and-azure-netapp-files-comparison"></a>Porównanie Azure Files i Azure NetApp Files

Ten artykuł zawiera porównanie Azure Files i Azure NetApp Files. 

Większość obciążeń, które wymagają magazynu plików w chmurze, na Azure Files lub Azure NetApp Files. Aby pomóc w ustaleniu najlepszego dopasowania do obciążenia, przejrzyj informacje zawarte w tym artykule. Aby uzyskać więcej informacji, zobacz dokumentację [Azure Files](https://docs.microsoft.com/azure/storage/files/) i [Azure NetApp Files](https://docs.microsoft.com/azure/azure-netapp-files/) .
