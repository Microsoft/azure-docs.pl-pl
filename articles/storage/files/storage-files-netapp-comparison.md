# <a name="azure-files-and-azure-netapp-files-comparison"></a>Azure Files i Azure NetApp Files porównanie

Ten artykuł zawiera porównanie Azure Files i Azure NetApp Files. 

Większość obciążeń, które wymagają magazynu plików w chmurze, działa dobrze Azure Files lub Azure NetApp Files. Aby ułatwić określenie najlepszego dopasowania do obciążenia, zapoznaj się z informacjami w tym artykule. Aby uzyskać więcej informacji, zobacz [Azure Files](https://docs.microsoft.com/azure/storage/files/) i [Azure NetApp Files](https://docs.microsoft.com/azure/azure-netapp-files/) dokumentacji.
