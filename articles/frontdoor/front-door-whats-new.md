# <a name="whats-new-in-azure-front-door"></a>Co nowego w funkcjach frontonu platformy Azure?

Drzwi frontonu platformy Azure są regularnie aktualizowane. Aby zaktualizować najnowsze zmiany, ten artykuł zawiera informacje na temat:

- Najnowsze wersje
- Znane problemy
- Poprawki błędów
- Funkcje uznane za przestarzałe
