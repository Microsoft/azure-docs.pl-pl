# <a name="getting-started-with-azure-iot-embedded-device-development"></a>Wprowadzenie do tworzenia urządzeń osadzonych w usłudze Azure IoT

**Dotyczy:** [Opracowywanie urządzeń osadzonych](about-iot-develop.md#embedded-device-development)

Ten przewodnik z wprowadzeniem zawiera zestaw przewodników Szybki start, które pokazują, jak rozpocząć pracę z urządzeniami osadzonymi i usługą Azure IoT. 

W każdym przewodniku Szybki start wykonaj następujące podstawowe zadania:
* Instalowanie zestawu osadzonych narzędzi programistycznych do programowania określonego urządzenia w języku C
* Tworzenie obrazu, który zawiera Azure RTOS i przykłady, a następnie flashuje urządzenie
* Bezpieczne łączenie urządzenia z usługą Azure IoT
* Wyświetlanie telemetrii urządzenia, wyświetlanie właściwości i wywoływanie metod chmura-urządzenie
