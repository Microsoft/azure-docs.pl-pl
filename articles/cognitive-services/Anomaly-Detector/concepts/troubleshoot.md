# <a name="troubleshooting-the-multivariate-api"></a>Rozwiązywanie problemów z interfejsem API wieloczynnikowa

Ten artykuł zawiera wskazówki dotyczące rozwiązywania problemów i korygowania typowych komunikatów o błędach HTTP podczas korzystania z interfejsu API wieloczynnikowa.
