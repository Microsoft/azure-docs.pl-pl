## <a name="prerequisites"></a>Wymagania wstępne

* Użyj subskrypcji platformy Azure. Jeśli nie masz subskrypcji platformy Azure, przed rozpoczęciem utwórz [bezpłatne konto][free-account].
* Zainstaluj [interfejs wiersza polecenia platformy Azure][azure-cli] (az).
* [Aparat platformy Docker][docker-engine] i sprawdzanie poprawności działania interfejsu wiersza polecenia platformy Docker w oknie konsoli.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: /cli/azure/install-azure-cli
[docker-engine]: https://www.docker.com/products/docker-engine
