## <a name="get-intent-programmatically"></a>Pobieranie intencji w sposób programistyczny

Użyj języka Java, aby wykonać zapytanie dotyczące [punktu końcowego przewidywania](https://aka.ms/luis-apim-v3-prediction) i uzyskać wynik przewidywania.

1. Utwórz nowy folder do przechowywania projektu Java, np `java-predict-with-rest` ..

1. Utwórz podkatalog o nazwie `lib` i skopiuj w następującym libs języka Java do `lib` podkatalogu:

    * [commons-logging-1.2.jar](https://raw.githubusercontent.com/Azure-Samples/cognitive-services-language-understanding/master/documentation-samples/quickstarts/analyze-text/java/lib/commons-logging-1.2.jar)
    * [httpclient-4.5.3.jar](https://raw.githubusercontent.com/Azure-Samples/cognitive-services-language-understanding/master/documentation-samples/quickstarts/analyze-text/java/lib/httpclient-4.5.3.jar)
    * [httpcore-4.4.6.jar](https://raw.githubusercontent.com/Azure-Samples/cognitive-services-language-understanding/master/documentation-samples/quickstarts/analyze-text/java/lib/httpcore-4.4.6.jar)

1. Skopiuj następujący kod, aby utworzyć klasę w pliku o nazwie `Predict.java`:

    [!code-java[Code snippet](~/cognitive-services-quickstart-code/java/LUIS/java-predict-with-rest/Predict.java)]

1. Zastąp wartości zaczynające się od `YOUR-` własnymi wartościami.

    |Informacje|Przeznaczenie|
    |--|--|
    |`YOUR-APP-ID`|Identyfikator aplikacji. Na stronie Ustawienia aplikacji Portal LUIS dla aplikacji.
    |`YOUR-PREDICTION-KEY`|Klucz predykcyjny dla 32 znaków. Znajdujący się w portalu LUIS, stronie zasobów platformy Azure dla Twojej aplikacji.
    |`YOUR-PREDICTION-ENDPOINT`| Punkt końcowy adresu URL przewidywania. Znajdujący się w portalu LUIS, stronie zasobów platformy Azure dla Twojej aplikacji.<br>Na przykład `https://westus.api.cognitive.microsoft.com/`.|

1. Kompiluj program Java z wiersza polecenia.

    * Jeśli używasz systemu Windows, użyj tego polecenia: `javac -cp ";lib/*" Predict.java`
    * Jeśli używasz macOS lub Linux, użyj tego polecenia: `javac -cp ":lib/*" Predict.java`

1. Uruchom program Java z wiersza polecenia:

    * Jeśli używasz systemu Windows, użyj tego polecenia: `java -cp ";lib/*" Predict`
    * Jeśli używasz macOS lub Linux, użyj tego polecenia: `java -cp ":lib/*" Predict`
