# <a name="backup-solutions-for-azure-vmware-solution-virtual-machines-vms"></a>Rozwiązania do tworzenia kopii zapasowych Azure VMware Solution maszyn wirtualnych

Kluczową zasadą Azure VMware Solution jest umożliwienie dalszego korzystania z inwestycji i ulubionych rozwiązań VMware działających na platformie Azure. Ważnym elementem tej strategii jest obsługa technologii niezależnego dostawcy oprogramowania (ISV Azure VMware Solution zweryfikowana za pomocą Azure VMware Solution oprogramowania. 

Nasi partnerzy ds. kopii zapasowych mają wiodące w branży rozwiązania do tworzenia kopii zapasowych i przywracania w środowiskach opartych na programie VMware. Klienci powszechnie przyjmują te rozwiązania dla swoich wdrożeń lokalnych. Teraz ci partnerzy rozszerzyli swoje rozwiązania na Azure VMware Solution, używając platformy Azure do zapewnienia repozytorium kopii zapasowych i miejsca docelowego magazynu na potrzeby długoterminowego przechowywania i archiwizowania.

Tworzenie kopii zapasowej ruchu sieciowego między maszynami Azure VMware Solution i repozytorium kopii zapasowych na platformie Azure odbywa się za pośrednictwem łącza o wysokiej przepustowości i małych opóźnieniach. Ruch replikacji między regionami odbywa się za pośrednictwem wewnętrznej sieci montażowej platformy Azure, co obniża koszty przepustowości dla użytkowników.
