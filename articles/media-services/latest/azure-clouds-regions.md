# <a name="azure-media-services-v3-clouds-and-regions-availability"></a>Dostępność chmur Azure Media Services i regionów w wersji 3

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

Azure Media Services wersja 3 jest dostępna za pośrednictwem Azure Resource Manager. Jednak nie wszystkie funkcje Media Services są dostępne we wszystkich chmurach platformy Azure. W tym dokumencie przedstawiono dostępność głównych składników Media Services w wersji 3. W poniższych tabelach przedstawiono, które funkcje Media Services są dostępne w poszczególnych regionach.
