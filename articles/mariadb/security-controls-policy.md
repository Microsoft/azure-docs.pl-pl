# <a name="azure-policy-regulatory-compliance-controls-for-azure-database-for-mariadb"></a>Azure Policy kontroli zgodności z przepisami dla Azure Database for MariaDB

[Zgodność z przepisami](../governance/policy/concepts/regulatory-compliance.md) w Azure Policy udostępnia definicje inicjatyw utworzonych przez firmę  Microsoft i  zarządzanych, znane jako _wbudowane,_ dla domen zgodności i kontroli zabezpieczeń związanych z różnymi standardami zgodności. Na tej stronie **wymieniono domeny zgodności i** **mechanizmy kontroli zabezpieczeń** dla Azure Database for MariaDB. Możesz indywidualnie przypisać wbudowane elementy  kontroli zabezpieczeń, aby zapewnić zgodność zasobów platformy Azure z określonym standardem.
