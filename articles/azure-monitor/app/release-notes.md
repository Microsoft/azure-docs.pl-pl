# <a name="release-notes---application-insights"></a>Informacje o wersji — Application Insights

Ta strona zawiera szczegółowe informacje o aktualizacjach i poprawkach błędów dla poszczególnych zestawów SDK Application Insights.

* Zestawy SDK platformy .NET
    - W przypadku wersji 2,12 i nowszej: [zestawy SDK platformy .NET (w tym ASP.NET, ASP.NET Core i karty rejestrowania)](https://github.com/Microsoft/ApplicationInsights-dotnet/releases)
