# <a name="java-codeless-application-monitoring-on-premises---azure-monitor-application-insights"></a>Monitorowanie aplikacji bezkodowej w języku Java Azure Monitor lokalnego Application Insights

Monitorowanie aplikacji bez kodu Java ma wszystkie informacje o prostotie — nie ma żadnych zmian w kodzie, a agent Java można włączyć za pomocą tylko kilku zmian konfiguracji.

## <a name="overview"></a>Omówienie

Po włączeniu Agent Java będzie automatycznie zbierać wiele żądań, zależności, dzienniki i metryki z najczęściej używanych bibliotek i struktur.

Postępuj zgodnie [ze szczegółowymi instrukcjami](./java-in-process-agent.md) dla wszystkich środowisk, w tym lokalnie.
