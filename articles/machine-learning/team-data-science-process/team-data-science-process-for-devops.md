# <a name="team-data-science-process-for-developer-operations"></a>Proces nauki danych zespołu dla operacji deweloperskich

W tym artykule przedstawiono funkcje deweloperskie (DevOps), które są specyficzne dla implementacji zaawansowanej analizy i rozwiązania Cognitive Services. Te materiały szkoleniowe implementują proces nauki o danych zespołowych (przetwarzania TDSP) oraz oprogramowanie i zestawy narzędzi firmy Microsoft i typu open source, pomocne w przypadku planowanie, wykonywania i dostarczania rozwiązań do nauki o danych. Odwołuje się do tematów obejmujących DevOps łańcucha narzędzi, które są specyficzne dla projektów i rozwiązań z zakresu nauki i przetwarzania danych.
