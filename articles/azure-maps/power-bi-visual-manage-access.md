# <a name="manage-the-azure-maps-visual-within-your-organization"></a>Zarządzanie wizualizacją Azure Maps w organizacji

Power BI zapewnia projektantom i administratorom dzierżawy możliwość zarządzania użyciem wizualizacji Azure Maps. Poniżej znajdują się kroki, które mogą wykonać poszczególne role.

### <a name="designer-options"></a>Opcje projektanta

W Power BI Desktop projektanci mogą wyłączyć wizualizację Azure Maps na karcie Zabezpieczenia. Wybierz  &gt; **Opcje pliku i ustawienia** , a następnie wybierz **Opcje** &gt; **Podgląd funkcji**. Po wyłączeniu Azure Maps nie zostanie domyślnie załadowana.  
  
![Panel opcje Power BI wyświetla opcje funkcji Podgląd Azure Maps wizualizacji](media/power-bi-visual/preview-options-panel.png)

## <a name="tenant-admin-options"></a>Opcje administratora dzierżawy

W programie PowerBI.com Administratorzy dzierżawy mogą wyłączyć wizualizację Azure Maps dla wszystkich użytkowników. Wybierz pozycje **Ustawienia** ustawienia &gt;   &gt; **dzierżawy** portalu administracyjnego. Gdy ta wartość jest wyłączona, Power BI nie będzie już wyświetlać wizualizacji Azure Maps w okienku wizualizacje.

> [!div class="mx-imgBorder"]
> ![Portal administracyjny Power BI przedstawiający ustawienia dzierżawy dla wizualizacji Azure Maps](media/power-bi-visual/tenant-admin-settings.png)
