# <a name="azure-policy-regulatory-compliance-controls-for-azure-rbac"></a>Azure Policy kontroli zgodności z przepisami dla kontroli RBAC platformy Azure

[Azure Policy](../governance/policy/index.yml) wymuszać reguły dla zasobów platformy Azure, aby infrastruktura spełniała standardy biznesowe.
[Zgodność z przepisami](../governance/policy/concepts/regulatory-compliance.md) w Azure Policy udostępnia definicje inicjatyw utworzonych przez firmę  Microsoft i  zarządzanych, znane jako _wbudowane,_ dla domen zgodności i kontroli zabezpieczeń związanych z różnymi standardami zgodności. Na tej stronie **wymieniono domeny zgodności i** **mechanizmy kontroli zabezpieczeń** dla kontroli dostępu na podstawie ról (RBAC) platformy Azure. Możesz indywidualnie przypisać wbudowane elementy  kontroli zabezpieczeń, aby zapewnić zgodność zasobów platformy Azure z określonym standardem.
