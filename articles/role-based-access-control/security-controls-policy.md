# <a name="azure-policy-regulatory-compliance-controls-for-azure-rbac"></a>Azure Policy kontroli zgodności z przepisami dla kontroli RBAC platformy Azure

[Azure Policy](../governance/policy/index.yml) wymuszać reguły dla zasobów platformy Azure, aby infrastruktura spełniała standardy biznesowe.
[Zgodność z przepisami](../governance/policy/concepts/regulatory-compliance.md) w Azure Policy udostępnia definicje inicjatyw utworzonych przez firmę  Microsoft i  zarządzanych, znane jako _wbudowane,_ dla domen zgodności i kontroli zabezpieczeń związanych z różnymi standardami zgodności. Ta strona zawiera listę **domen zgodności i** kontroli zabezpieczeń **dla** kontroli dostępu na podstawie ról (RBAC) platformy Azure. Możesz przypisać wbudowane elementy kontroli  zabezpieczeń indywidualnie, aby ułatwić zgodność zasobów platformy Azure z określonym standardem.
