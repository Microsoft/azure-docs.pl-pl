# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>Wprowadzenie do funkcji poznawczej języka U-SQL

## <a name="overview"></a>Omówienie
Możliwości poznawcze dla języka U-SQL umożliwiają deweloperom korzystanie z funkcji przesyłania danych Big Data w programach. 

Dostępne są następujące przykłady korzystające z możliwości poznawczej:
* Imaging: [wykrywanie](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world) twarzy
* Imaging: [wykrywanie rozpoznawania emocji](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Imaging: [wykrywanie obiektów (tagowanie)](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Imaging: [OCR (optyczne rozpoznawanie znaków)](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Tekst: [wyodrębnianie kluczowych fraz & analiza tonacji](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)
