# <a name="overview-of-the-irs-1075-blueprint-sample"></a>Omówienie przykładowej strategii IRS 1075

Przykład strategii IRS 1075 udostępnia wytyczne dotyczące ładu w usłudze [Azure Policy](../../../policy/overview.md), które ułatwiają ocenę określonych mechanizmów kontroli IRS 1075. Ta strategia ułatwia klientom wdrażanie podstawowego zestawu zasad dla dowolnej architektury wdrożonej przez platformę Azure, która musi implementować mechanizmy kontroli IRS 1075.

## <a name="control-mapping"></a>Mapowanie kontrolek

Sekcja mapowania mechanizmów kontroli zawiera szczegółowe informacje dotyczące zasad w ramach tej strategii i o tym, jak te zasady pomagają spełnić wymagania różnych mechanizmów kontroli zgodnie ze standardem IRS 1075. Po przypisaniu do architektury zasoby są oceniane przez usługę Azure Policy pod względem braku zgodności z przypisanymi zasadami. Aby uzyskać więcej informacji, zobacz [Azure Policy](../../../policy/overview.md).
