# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Wyświetlanie spisu wystąpień w Azure Portal

Po przekazaniu [metryk, dzienników](upload-metrics-and-logs-to-azure-monitor.md)lub [użycia](view-billing-data-in-azure.md)można wyświetlić wystąpienie jako zasób platformy Azure z Azure Portal. Aby wyświetlić zasób w Azure Portal, przejdź do specjalnego adresu URL [https://portal.azure.com](https://portal.azure.com) i wykonaj następujące kroki:

1. Przejdź do pozycji **wszystkie usługi**.
1. Wyszukaj typ wystąpienia bazy danych.
1. Dodaj typ do ulubionych.
1. W menu po lewej stronie wybierz typ wystąpienia.
1. Wyświetlaj wystąpienia w tym samym widoku co inne zasoby usługi Azure SQL lub Azure PostgreSQL w skali (Użyj filtrów dla widoku szczegółowego).
