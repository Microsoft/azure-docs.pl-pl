# <a name="azure-active-directory-connect-health-alert-catalog"></a>Azure Active Directory Connect Health wykaz alertów 

Alerty wysyłania usługi Azure AD Connect Health wskazują, że infrastruktura tożsamości nie jest w dobrej kondycji. Ten artykuł zawiera tytuły i opisy alertów oraz czynności zaradcze dla każdego alertu. <br />
Error, Warning i prewarning to trzy etapy alertów, które są generowane na podstawie usługi Connect Health. Zdecydowanie zalecamy podejmowanie natychmiastowych akcji dotyczących alertów wyzwalanych. <br />
Alerty Azure AD Connect Health są rozwiązywane po pomyślnym spełnieniu warunku. Azure AD Connect Health agenci wykrywają i raportują warunki sukcesu usługi okresowo. W przypadku kilku alertów pomijanie jest oparte na czasie. Innymi słowy, jeśli ten sam warunek błędu nie zostanie zaobserwowany w ciągu 72 godzin od wygenerowania alertu, alert zostanie automatycznie rozwiązany.
