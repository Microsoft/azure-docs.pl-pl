# <a name="deprecation-of-azure-service-manager-support-for-azure-service-bus-relay-and-event-hubs"></a>Obsługa usług Azure Service Manager dla Azure Service Bus, Relay i Event Hubs

Resource Manager, nasz stos infrastruktury chmurowej nowej generacji, całkowicie zastępuje "klasyczny" model zarządzania usługami platformy Azure (klasyczny model wdrażania). W związku z tym interfejsy API REST klasycznego modelu wdrażania i obsługa usług Service Bus, Relay i Event Hubs zostaną wycofane 1 listopada 2021 r. To cofanie obsługi zostało ogłoszone po raz pierwszy w komunikacie społeczności technicznej firmy [Microsoft,](https://techcommunity.microsoft.com/t5/Service-Bus-blog/Deprecating-Service-Management-support-for-Azure-Service-Bus/ba-p/370909)ale niedawno zdecydowaliśmy się na przedłużenie okresu obsługi o kolejne lata od czasu pierwotnego ogłoszenia. Aby ułatwić identyfikację, te interfejsy API `management.core.windows.net` mają identyfikator URI. Zapoznaj się z następującą tabelą, aby uzyskać listę przestarzałych interfejsów API i ich Azure Resource Manager, których należy teraz używać.

Aby nadal używać Service Bus, usługi Relay i Event Hubs, przejdź do Resource Manager do 31 października 2021 r. Zachęcamy wszystkich klientów, którzy nadal korzystają ze starych interfejsów API, aby wkrótce dokonali przełączenia, aby skorzystać z dodatkowych korzyści usługi Resource Manager, takich jak grupowanie zasobów, tagi, usprawniony proces wdrażania i zarządzania oraz skoja fazie precyzyjnej kontroli dostępu przy użyciu kontroli dostępu na podstawie ról (RBAC) platformy Azure.

Aby uzyskać więcej informacji na temat Azure Resource Manager vs Azure Service Manager, zobacz [blog TechNet](https://blogs.technet.microsoft.com/meamcs/2016/12/22/difference-between-azure-service-manager-and-azure-resource-manager/).

Aby uzyskać więcej informacji na temat interfejsów API Service Manager i Resource Manager dla aplikacji Azure Service Bus, Relay i Event Hubs, zobacz dokumentację interfejsu API REST:

- [Azure Service Bus](/rest/api/servicebus/)
- [Azure Event Hubs](/rest/api/eventhub/)
- [Azure Relay](/rest/api/relay/)
