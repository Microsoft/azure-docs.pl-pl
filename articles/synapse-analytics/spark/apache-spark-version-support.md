# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>Obsługiwane wersje języka i środowiska uruchomieniowego dla Apache Spark i składników zależnych  

Apache Spark w usłudze Azure Synapse Analytics obsługuje kilka różnych czasów uruchamiania i usług ten dokument zawiera listę wersji.

## <a name="spark-version"></a>Wersja platformy Spark

Platforma Spark 2,4 jest aktualnie obsługiwaną wersją.

## <a name="operating-system-version"></a>Wersja systemu operacyjnego

Apache Spark w usłudze Azure Synapse działa w Ubuntu w wersji 16,04.

## <a name="java-version"></a>Wersja środowiska Java

Apache Spark w usłudze Azure Synapse obejmuje język Java w wersji 1.8.0 _272

## <a name="scala-version"></a>Wersja Scala

Apache Spark w usłudze Azure Synapse obejmuje Scala 2.11.12

## <a name="net-core-version"></a>Wersja platformy .NET Core

Apache Spark w usłudze Azure Synapse obejmuje platformę .NET Core 3,1

## <a name="net-support-for-apache-spark"></a>Obsługa platformy .NET dla Apache Spark

Usługa Azure Synapse obejmuje obsługę platformy .NET dla Apache Spark 1.0.0

## <a name="delta-lake"></a>Delta Lake

Synapse Analytics obejmuje obsługę różnicowej usługi Lake 0.6.1

## <a name="python-version"></a>Wersja języka Python

Apache Spark w usłudze Azure Synapse obejmuje instalację języka Python w wersji 3.6.1
