# <a name="azure-policy-regulatory-compliance-controls-for-azure-kubernetes-service-aks"></a>Azure Policy kontroli zgodności z przepisami dla usługi Azure Kubernetes Service (AKS)

[Zgodność z przepisami w Azure Policy](../governance/policy/concepts/regulatory-compliance.md) zawiera definicje inicjatyw (*wbudowane*) utworzone i zarządzane przez firmę Microsoft, w przypadku domen zgodności i kontroli zabezpieczeń związanych z różnymi standardami zgodności. Ta strona zawiera listę domen zgodności usługi Azure Kubernetes Service (AKS) i kontroli zabezpieczeń.

Wbudowane elementy **kontroli zabezpieczeń** można przypisać indywidualnie, aby zapewnić zgodność zasobów platformy Azure z określonym standardem.
