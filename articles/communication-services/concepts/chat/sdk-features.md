# <a name="chat-sdk-overview"></a>Omówienie zestawu SDK rozmowy 

Zestawy SDK rozmowy z usługami Azure Communication Services mogą służyć do dodawania rozbudowanej, w czasie rzeczywistym rozmowy do aplikacji.
