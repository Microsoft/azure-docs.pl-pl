# <a name="azure-guest-os"></a>System operacyjny gościa platformy Azure
W poniższych tabelach przedstawiono Centrum zabezpieczeń firmy Microsoft (MSRC) zastosowane do systemu operacyjnego gościa platformy Azure. Przeszukaj ten artykuł, aby ustalić, czy określonej aktualizacji dotyczy systemu operacyjnego gościa, z których korzystasz. Aktualizacje są zawsze przekazywane dla określonej [rodziny,][family-explain] w których zostały wprowadzone.

## <a name="april-2021-guest-os"></a>System operacyjny gościa z kwietnia 2021 r.

>[!NOTE]

Po zakończeniu tego procesu ta wersja będzie dostępna dla aktualizacji ręcznych za pośrednictwem plików Azure Portal konfiguracji.
