# <a name="azure-iot-hub-get-started-with-physical-devices-tutorials"></a>Azure IoT Hub wprowadzenie do samouczków dotyczących urządzeń fizycznych

Te samouczki zawierają wprowadzenie do usługi Azure IoT Hub i zestawów SDK urządzeń. Samouczki obejmują typowe scenariusze IoT, aby zademonstrować możliwości IoT Hub. Samouczki ilustrują również sposób łączenia IoT Hub z innymi usługami i narzędziami platformy Azure w celu tworzenia bardziej wydajnych rozwiązań IoT. W samouczkach wymienionych w poniższej tabeli przedstawiono sposób tworzenia fizycznych urządzeń IoT.
